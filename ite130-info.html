Vad är Git?

Git är ett versionshanteringssystem som används för att spara, spåra och samarbeta kring kod. Med Git kan du enkelt se ändringar, gå tillbaka till tidigare versioner och arbeta tillsammans med andra i projekt. Git används ofta tillsammans med plattformar som GitHub och GitLab.

  • Versionshantering: Spara och återställ kodändringar.
  • Samarbete: Flera personer kan arbeta på samma projekt.
  • Historia: Se vem som gjort vad och när.
