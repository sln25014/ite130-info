# Vibe coding

Vibe coding är att med hjälp av AI skapa generativ kod via prompter eller få förslag och autokomplettera kod under tiden den skrivs. Vi kommer gå igenom hur man kommer igång med vibe coding och exempel på kod man kan skapa med hjälp av detta.

## Inline

Vibe coding kallas för **Inline** när AI som hjälpmedel används direkt när man skriver kod.

I visual studio code kan man få automatiska förslag för kod. Ett exempel kan vara om man skriver: ```<!DOCTYPE html>``` kan man få förslaget:

![Ghost text](bilder/html-completion.png)

Där man kan trycka ```Tab``` för att godkänna eller välja förslaget.

## Agenter

Att skriva 

För att öppna AI chatten klickar man på dropdown-menyn högst upp i visual studio code och sedan på `Open Chat` eller kortkommandot `Ctrl+Alt+I`.

![Hur man öppnar chatten](bilder/open_chat.png)

Till höger öppnas ett nytt fönster för att skriva prompter och ändra inställningar för AI botten, se bild:

![Chatruta](bilder/chatruta.png)

Man kan ändra vilken funktion man vill att AI botten ska ha genom att trycka på ```Agent``` för att få upp följande:

![Agenter](bilder/agents.png)

### Agent

Agent versionen är optimerad för komplicerade uppgifter och kan välja vilken fil, bland flera, som ska ändras och planerar arbete.

### Plan

Plan används för att strukturera arbete och att bryta ner komplexa uppgifter till mindre steg för lättare arbete. Att skapa en förståelse för problemet ligger i fokus.

### Ask

Ask är till för att svara på frågor relaterade till din kod, kodningsfrågor och generella teknologikoncept. Den används för förståelse och information snarare än att ändra kod

### Edit

Edit är specifikt till för att ändra filer i ett projekt och passar bäst när man har förståelse för vad som ska göras.
